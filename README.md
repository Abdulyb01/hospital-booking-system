# Hospital Booking and Follow-up Reminder System

A comprehensive full-stack web application for managing hospital appointments, patient bookings, and automated follow-up reminders.

## Features

✅ **Patient Management**
- Patient registration and profile management
- Medical history tracking
- Contact information management

✅ **Appointment Booking**
- Real-time appointment scheduling
- Doctor availability management
- Department-wise appointment booking

✅ **Doctor/Staff Management**
- Doctor profiles and specializations
- Staff scheduling
- Availability management

✅ **Automated Reminders**
- Email notifications
- SMS alerts
- Push notifications
- Customizable reminder timing

✅ **Follow-up System**
- Automated follow-up scheduling
- Follow-up status tracking
- Patient feedback collection

✅ **Calendar Integration**
- Visual appointment calendar
- Real-time availability view
- Conflict detection

✅ **Responsive Dashboard**
- Bootstrap-powered responsive UI
- Admin dashboard
- Patient dashboard
- Doctor dashboard

## Tech Stack

- **Backend:** PHP
- **Frontend:** HTML5, CSS3, JavaScript, Bootstrap 5
- **Database:** MySQL (phpMyAdmin)
- **Server:** Apache/Nginx

## Installation

See [INSTALLATION.md](./INSTALLATION.md) for detailed setup instructions.

## Getting Started

1. Clone the repository
2. Set up MySQL database using `database/schema.sql`
3. Configure database connection in `backend/config/db.php`
4. Place project in web server directory
5. Access via browser
